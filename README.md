# 📊 Customer Behavior Analysis Using SQL, Python & Power BI

## 🚀 Project Overview

This project analyzes customer shopping behavior using transactional retail data to uncover purchasing patterns, customer preferences, revenue trends, and loyalty insights.

The project demonstrates an end-to-end data analytics workflow including:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL-Based Business Analysis
- Customer Segmentation
- Power BI Dashboard Development
- Business Recommendations

The goal is to transform raw customer transaction data into actionable business insights that support strategic decision-making.

---

## 🎯 Business Problem

Retail and e-commerce companies collect large volumes of customer transaction data. However, identifying valuable insights from this data can be challenging.

Key business challenges include:

- Identifying high-value customers
- Understanding purchasing behavior
- Measuring subscription effectiveness
- Improving customer retention
- Optimizing inventory management
- Increasing revenue through targeted strategies

This project addresses these challenges through data-driven analysis.

## 📂 Dataset Overview

The dataset contains customer shopping transaction records including:

| Category | Description |
|-----------|------------|
| Customer Information | Age, Gender, Location |
| Product Information | Category, Item Purchased |
| Transaction Details | Purchase Amount, Shipping Type |
| Customer Feedback | Review Rating |
| Marketing Data | Subscription Status, Discounts |
| Purchase Behavior | Previous Purchases, Frequency |
| Seasonal Data | Season |

## 📈 Key Insights

### Customer Insights

- Loyal customers generate significant revenue.
- Subscribers spend more than non-subscribers.
- Repeat customers contribute higher lifetime value.

### Product Insights

- A small number of products generate the majority of sales.
- High-rated products show stronger performance.
- Product demand varies by season.

### Revenue Insights

- Customer segmentation creates opportunities for targeted marketing.
- Seasonal purchasing patterns impact revenue.
- Subscription programs improve customer engagement.
